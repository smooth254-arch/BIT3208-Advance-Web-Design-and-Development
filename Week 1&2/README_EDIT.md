# Project Editing Guide

This project is fully editable in VS Code. You can open any file in the workspace and make changes directly.

## How to edit files

1. Open the file you want to change from the Explorer panel.
2. Edit the code in the editor.
3. Save the file with `Ctrl+S`.
4. If you want, use the terminal to run or test the project.

## Files you can edit

- `index.php`, `login.php`, `register.php`, `Chichi.php`, `shop.php`, etc.
- `week 3/` and `Week 4/` folders
- `style.css` and any image or backend file

## Notes

- This workspace is not locked. Feel free to make code changes by yourself.
